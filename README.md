# BUP UCAM Marks Notifier [WIP]

This Project aims to provide real time notification of Uploaded/Updated Marks via Selenium.

## Installation

### 1. Make sure you have Python installed on your device

### 2. Clone the repository

```bash
git clone https://github.com/24phyr/bup-ucam-marks-notifier.git
cd bup-ucam-marks-notifier
```

### 3. Create and activate virtual environment

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 4. install all required dependencies

```bash
pip install -r requirements.txt
```

### Refer the [.env.example](https://github.com/24phyr/bup-ucam-marks-notifier/blob/main/.env.example) to set your .env file up
